- Create a very detailed todo list from the plan
- Execute the todo list

Todo list requirements:

- Todo list must have all the validation steps (e.g. npm run all-in)
- Todo list must have a convention validation step (e.g. validate the changes against the @steering-documents)

Execution requirements:

- The model must work on a single todo item at a time
- The model is not allowed to work on multiple todo items at the same time
- When a new problem is detected, the model must create a new todo item for it
